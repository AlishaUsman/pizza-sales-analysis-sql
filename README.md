# 🍕 Pizza Sales Analysis (SQL Project)

> End-to-end SQL analysis of 48,000+ pizza sales records to uncover meaningful business insights and patterns.

---

## 📌 Overview
This project focuses on analyzing pizza sales data using SQL to understand business performance and customer behavior.

---

## 🎯 What This Project Explores

| Area | Description |
|------|------------|
| 📊 Sales Analysis | Total orders and overall revenue trends |
| 🕒 Time Analysis | Peak ordering hours and customer activity patterns |
| 🍕 Product Performance | Best-performing pizzas and categories |
| 📦 Category Analysis | Category-wise sales distribution |
| 📈 Revenue Trends | Average daily orders and cumulative revenue |
| 🏆 Ranking | Top-performing pizzas using advanced SQL |

✔ A total of **13 SQL queries** were designed and executed to explore these insights.

---

## 🛠 Tools & Technologies

| Tool | Purpose |
|------|--------|
| MySQL | Database management & querying |
| SQL | Data analysis & transformation |

---

## 📂 Dataset Structure

| Table | Description |
|------|------------|
| orders | Contains order date and time |
| order_details | Contains quantity of pizzas per order |
| pizzas | Contains pizza size and price |
| pizza_types | Contains pizza names and categories |

---

## ⚙️ Key SQL Concepts Used

- Joins (INNER JOIN, LEFT JOIN)  
- Aggregations (SUM, COUNT, AVG)  
- Subqueries  
- Group By operations  
- Window Functions (RANK, OVER)
